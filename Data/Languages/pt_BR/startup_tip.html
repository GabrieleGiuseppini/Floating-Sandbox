Bem vindo ao Floating Sandbox!

Antes de você começar, nós temos apenas algumas dicas básicas para você aprender sobre os controles básicos do jogo:

  • Para carregar um objeto vá para o menu Início e selecione Carregar Objeto
  • Para acessar os comandos da câmera vá para o menu dos Controles.
  • Para usar as ferramentas vá para o menu Ferramentas e selecione uma ferramenta
  • Para adicionar um NPC vá para o menu dos NPCs.
  • Para mudar para dia ou noite use as teclas Page Up/Page Down ou então vá para o menu dos Controles.
  • Para editar ou criar um objeto/navio vá para o Construtor de Navios.
  • Para mais ajuda vá para o menu Ajuda e selecione Ajuda

Curta o jogo!
