Bienvenido a Floating Sandbox!

Antes de empezar, tenemos unos consejos para ayudarte a jugar:

  • Para cargar un barco ve al menu de Archivos y selecciona Cargar un Barco
  • Para cambiar la vista ve al menu de Controles
  • Para usar herramientas ve al menu de Herramientas y selecciona una herramienta
  • Para añadir un NPC ve al menu NPCs
  • Para cambiar el tiempo del dia usa las teclas RePág/AvPág o usa el menu de Controls
  • Para editar o crear un barco ve al menu del Constructor de Barcos
  • Para mas ayuda entra el menu de Ayuda y selleciona Ayuda

Disfruta!
